# Task 1: Predicting House Prices with Linear Regression

Level: 2

## 📌 Project Overview
The objective of this project was to build a predictive machine learning model to estimate house prices based on various relevant features. By developing this model, I gained hands-on experience in implementing, evaluating, and interpreting a fundamental linear regression algorithm.

## 🛠️ Tech Stack
* **Python:** Scripting and logic.
* **Pandas & NumPy:** Data manipulation, cleaning, and text-to-numerical encoding.
* **Scikit-Learn:** Machine learning modeling, splitting training/testing sets, and performance evaluation.
* **Seaborn & Matplotlib:** Data visualization.

## 🧠 Machine Learning Process
1. **Data Preprocessing:** Inspected the dataset and converted text-based categorical variables (like "yes/no" and furnishing status) into numerical formats using binary mapping and dummy variables.
2. **Model Training:** Separated the dataset into features (area, bedrooms, etc.) and the target variable (price). Split the data into an 80% training set and a 20% testing set. 
3. **Implementation:** Trained a **Linear Regression** model using Scikit-Learn.
4. **Evaluation & Visualization:** The model successfully learned the relationships and achieved an **R-squared Score of 0.6529**. I generated a scatter plot using Matplotlib and Seaborn to visually compare the actual house prices against the model's predicted prices.
